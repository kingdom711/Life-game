# 프로젝트 진행 현황 보고서

> **프로젝트명**: 안전의 길 (Safety Quest Game)  
> **작성일**: 2024-12-20  
> **최종 업데이트**: 2026-01-08  
> **프로젝트 타입**: 게이미피케이션 기반 안전관리 플랫폼

---

## 📋 프로젝트 개요

**안전의 길**은 현장 안전관리를 자발적으로 모두가 참여하게 하는 게이미피케이션 플랫폼입니다. 중대재해처벌법 대응을 위한 법정 자율점검표 전산화, 역할 기반 워크플로우 자동화, AI 기반 위험도 분석 기능을 제공합니다.

### 핵심 가치 제안
- 법정 기준 체크리스트 전산화
- 역할 기반 워크플로우 자동화
- AI 기반 사진 위험도 분석
- "내 할 일" 중심 실무 UX

---

## 🛠 기술 스택

### 프론트엔드
- **프레임워크**: React 18.2.0
- **빌드 도구**: Vite 5.0.8
- **라우팅**: React Router DOM 6.20.0
- **스타일링**: 
  - Tailwind CSS 4.1.18 (최근 도입)
  - 커스텀 CSS (기존 스타일 유지)
- **언어**: JavaScript (ES6+)

### 백엔드 (예정)
- **프레임워크**: Spring Boot
- **AI 통합**: Google Gemini API (통합 준비 완료)

### 개발 환경
- **Node.js**: 18.x 이상
- **패키지 관리**: npm
- **개발 서버**: Vite Dev Server (포트 3000)

## 🎨 디자인 시스템

### 색상 시스템
- **안전 색상**: Safe (Green), Warning (Orange), Danger (Red)
- **기본 색상**: Primary (Blue), Secondary (Purple)
- **아이템 희귀도**: Common, Rare, Epic, Legendary

### 스타일링 접근법
- **하이브리드 방식**: Tailwind CSS + 커스텀 CSS
- **Glassmorphism**: 반투명 배경 + backdrop-blur
- **그라데이션**: 텍스트, 배경, 버튼에 적용
- **애니메이션**: Shimmer, Float, Fade-in 등

### 1. 레벨 시스템 (15단계)

- 범위: Bronze III (0P) ~ Diamond I (1,900,000P+)
- 설계 기준: 중간 활동량(150,000P/년) 기준 1년에 1단계 상승
- 핵심 파일: src/utils/pointsCalculator.js

- 매월 1일 자동 리셋
- 보상: 30P~2,000P + 아이템 상자
- 핵심 파일: src/utils/storage.js, src/components/StreakButton.jsx

### 3. 퀘스트 시스템
- 일일 퀘스트: 매일 자정 리셋
- 주간 퀘스트: 매주 월요일 리셋
- 월간 퀘스트: 매월 1일 리셋
- 역할별 퀘스트: technician(기술자), supervisor(관리감독자), safety_manager(안전관리자)
- 핵심 파일: src/data/questsData.js

### 4. 아이템/인벤토리 시스템 2.0
- 희귀도: Common, Rare, Epic, Legendary
- 카테고리: HELMET, VEST, GLOVES, SHOES, BELT, MASK, GLASSES
- 장비 장착 시 아바타에 반영
- **검교정(강화)**: 포인트 소모로 장비 레벨업, 스탯 증가
- **안전 스탯**: pointBoost(포인트 획득량%), xpAccelerator(경험치 가속%), streakSaver(스트릭 보호%)
- **세트 효과**: 동일 세트 2/4/7피스 장착 시 추가 보너스
- 핵심 파일: src/data/itemsData.js, src/data/setsData.js, src/utils/inventoryManager.js, src/utils/calibrationService.js
